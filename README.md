# 📊 Telegram-бот: сбор отчётов и расчёт ЗП

Бот для автоматизации ежедневных отчётов смены, расчёта зарплаты и формирования Excel-сводок.

---

## 🚀 Быстрый старт (Docker)

### 1. Узнайте свой Telegram ID

Напишите боту [@userinfobot](https://t.me/userinfobot) — он пришлёт ваш числовой ID.

### 2. Настройте `.env`

```env
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_IDS=ваш_telegram_id          # через запятую если несколько
ADMIN_CHAT_ID=                      # ID группы для дублей (опционально)
DATABASE_URL=sqlite+aiosqlite:///data/bot.db
```

### 3. Запустите через Docker

```bash
docker compose up -d --build
```

### 4. Проверьте логи

```bash
docker compose logs -f
```

## 👤 Роли пользователей

| Роль       | Возможности                       |
| ---------- | --------------------------------- |
| `pending`  | Заявка отправлена, ждёт одобрения |
| `employee` | Сдача отчётов, просмотр своей ЗП  |
| `admin`    | Всё выше + панель управления      |

**Первый вход:** новый пользователь пишет `/start` → отправляется заявка → администратор одобряет кнопкой прямо в чате.

**Быстрый старт без ожиданий:** добавьте свой ID в `ADMIN_IDS` в `.env` — вы автоматически получите права администратора.

## 💰 Шкала ЗП (по умолчанию)

| Уровень | Выручка         | Оклад   | % с выручки |
| ------- | --------------- | ------- | ----------- |
| 1       | до 15 000 ₽     | 2 500 ₽ | 10%         |
| 2       | 15 000–30 000 ₽ | —       | 20%         |
| 3       | от 30 000 ₽     | —       | 22%         |

ЗП делится поровну между всеми сотрудниками смены.  
Пороги настраиваются через **Админ-панель → Шкала ЗП**.

---

## 📋 Сдача отчёта (12 шагов)

1. Дата смены
2. Название проекта
3. Фамилия сотрудника
4. Кол-во человек в смене
5. Общая выручка
6. Наличные
7. Эквайринг (безнал)
8. Расход
9. Остаток в кассе
10. Проходимость (посетители)
11. Дней рождений
12. Комментарий (опционально)

→ Превью с расчётом ЗП → подтверждение → сохранение + уведомление администратору.

## ⚙️ Админ-панель

- **📊 Отчёты** — скачать Excel за текущий/прошлый месяц, год или произвольный период
- **👥 Сотрудники** — добавить по Telegram ID, назначить/снять права, удалить
- **📥 Заявки** — одобрить/отклонить новых пользователей
- **💰 Шкала ЗП** — редактировать уровни прогрессивной шкалы
- **🎯 Планы продаж** — добавлять планы по проекту (день/месяц)
