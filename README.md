# 📊 Telegram-бот: сбор отчётов и расчёт ЗП

Бот для автоматизации ежедневных отчётов смены, расчёта зарплаты, формирования Excel-сводок и визуальной аналитики.

---

## ✨ Новые возможности

- **🗄 PostgreSQL**: Переход на промышленную БД для надёжности.
- **📈 Визуальная аналитика**: Графики выручки и выполнения планов (Matplotlib).
- **👤 Личный кабинет**: Просмотр баланса, истории выплат и статистики сотрудника.
- **💸 Система выплат**: Учёт задолженностей, премий, штрафов и фиксация факта выплаты.
- **🔔 Напоминания**: Автоматические уведомления о необходимости сдать отчёт.
- **🏗 Миграции**: Версионирование базы данных через Alembic.

---

## 🚀 Быстрый старт (Docker)

### 1. Подготовка `.env`

Скопируйте шаблон и заполните его:

```bash
cp .env.example .env
```

**Основные параметры:**

- `BOT_TOKEN`: токен от [@BotFather](https://t.me/BotFather).
- `ADMIN_IDS`: ваш ID (узнать в [@userinfobot](https://t.me/userinfobot)).
- `DATABASE_URL`: `postgresql+asyncpg://user:password@db:5432/ph_time_tg`

### 2. Запуск

```bash
docker compose up -d --build
```

Бот автоматически применит миграции и создаст необходимые таблицы.

## 💰 Расчёт ЗП

Шкала настраивается в админ-панели.

- **Формула**: `ЗП = Оклад + (% с выручки / кол-во человек)`.
- **Оклад**: выплачивается в полном объёме каждому сотруднику в смене.
- **Процент**: делится поровну между всеми участниками смены.

## ⚙️ Админ-панель

- **📊 Отчёты (Excel)**: за день, месяц, год или свой период.
- **📈 Аналитика**: интерактивные графики (30 дней / Годовой тренд).
- **💸 Выплаты**: управление долгами, начисление премий и штрафов.
- **👥 Управление**: одобрение заявок, управление правами доступа.
- **🎯 Планы**: установка целей продаж по проектам.
